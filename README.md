Private Tracker back-end.

## 已实现功能

- 种子上传、下载
- 站内信系统
- 邮箱系统
- 种子举报
- 种子置顶、种子免费
- 用户积分（Coin）获取、消费
- 公告系统
- 用户等级系统
- 配置系统
- 缓存系统（目前采用 Garnet）
- 种子搜索
- 种子请求（愿望单）
- 种子列表排序（按文件大小、上传人数、下载人数）
- 通过 TMDb 来获取信息（封面、简介等信息）

## 未实现

- 为管理员提供一个界面来管理用户、公告、举报等

## 使用到的开源库/鸣谢

- MeiliSearch
- gemini cli
- PostgreSql
